# Prerequisites
 * GCC with C++17 support
 * Cmake >=3.12
 * GNU Make

# How to build
Manually:
```
$ mkdir cmake_build
$ cd cmake_build
$ cmake ..
$ make
$ cd ..
```

With shortcut:
```
$ make build_release
```
